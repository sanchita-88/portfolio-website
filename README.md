# Sarthak Khare — Portfolio Website

A personal portfolio website built with React, Three.js, and Tailwind CSS.

🔗 **Live:** [portfolio-website-roan-mu-22.vercel.app](https://portfolio-website-roan-mu-22.vercel.app)

---

## 🛠️ Tech Stack

React · TypeScript · Vite · Three.js · React Three Fiber · Tailwind CSS

---

## 🚀 Running Locally

```sh
git clone https://github.com/sanchita-88/portfolio-website.git
cd portfolio-website
npm install
npm run dev
```
